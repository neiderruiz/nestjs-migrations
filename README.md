# migrations nestjs

## clone project

```bash
git clone https://github.com/neiderruiz/nestjs-migrations.git
```

## inner project

```bash
cd nestjs-migrations
```

## install dependencies

```bash
npm install
```

## create migration

```bash
npm run migration:generate --name=example_name
```

## run migrations

```bash
npm run migration:run
```

## revert migrations

```bash
npm run migration:revert
```
